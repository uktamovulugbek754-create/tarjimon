### 1️⃣ Demo (30 soniya)
```bash

```

### 2️⃣ Haqiqiy (Google Cloud)
```bash

venv\Scripts\activate
pip install -r requirements.txt
set GOOGLE_APPLICATION_CREDENTIALS=credentials.json
python app.py
```

### 3️⃣ Docker
```bash
docker-compose up
```

## 🚀 Production Deployment

Tayyor bo'lganingizda:

```bash
# 1. Environment o'rnating
set FLASK_ENV=production

# 2. WSGI server o'rnating
pip install gunicorn

# 3. Ishga tushirish
gunicorn -w 4 -b 0.0.0.0:5000 app:app

# 4. Reverse proxy (Nginx) konfiguratsiyalash
# 5. SSL certificate qo'shish
# 6. Monitoring setup qilish
# 7. Backups avtomatik qilish
```
